# Izrada tabele neodređenih integrala u programu Mathematica

## Povzetek projektne naloge
V okviru projektne naloge bom s programom Mathematica izdelal pregledno tabelo standardnih neodređenih integralov. 
Cilj naloge je uporabiti simbolno računanje za izračun integralov različnih funkcij (polinomi, trigonometrične in eksponentne funkcije) ter rezultate izvoziti v obliki PDF dokumenta.
Projekt bo prikazal osnovno uporabo seznamov, funkcije `Integrate` in oblikovanje tabelaričnih podatkov.
